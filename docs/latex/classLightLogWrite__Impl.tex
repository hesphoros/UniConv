\doxysubsection{Detailed Description}
Lightweight asynchronous logging implementation. 

Thread-\/safe logging class that provides asynchronous log writing with support for multiple string encodings and automatic file rotation. Uses a dedicated writer thread and producer-\/consumer pattern for high-\/performance logging without blocking the calling threads.

\begin{DoxyParagraph}{Usage Pattern\+:}

\begin{DoxyEnumerate}
\item Create instance and optionally set log file or persistent logging
\item Call \doxylink{classLightLogWrite__Impl_a78a9cc299bb69aff33cab42d172a2841}{Write\+Log\+Content()} from any thread to add log entries
\item Destructor automatically handles cleanup and thread synchronization
\end{DoxyEnumerate}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Thread Safety\+:}

\begin{DoxyItemize}
\item All public methods are thread-\/safe
\item Uses mutex for queue operations and file access
\item Condition variable for efficient thread communication
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{File Rotation\+:}

\begin{DoxyItemize}
\item Automatic rotation based on AM/\+PM periods
\item Files named as\+: basename\+\_\+\+YYYY\+\_\+\+MM\+\_\+\+DD\+\_\+\+AM/\+PM.\+log
\item Directory structure auto-\/created as needed
\end{DoxyItemize}
\end{DoxyParagraph}
