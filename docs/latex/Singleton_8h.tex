\doxysubsection{Detailed Description}
Thread-\/safe \doxylink{classSingleton}{Singleton} pattern template implementation. 

This file provides a thread-\/safe, modern C++ implementation of the \doxylink{classSingleton}{Singleton} pattern using std\+::shared\+\_\+ptr and std\+::call\+\_\+once for guaranteed thread safety and lazy initialization.

\begin{DoxyParagraph}{Key Features\+:}

\begin{DoxyItemize}
\item Thread-\/safe lazy initialization using std\+::call\+\_\+once
\item RAII-\/compliant using std\+::shared\+\_\+ptr
\item Move-\/only semantics (no copy constructor/assignment)
\item Template-\/based for type safety
\item Memory-\/safe automatic cleanup
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Usage Pattern\+:}
Inherit from \doxylink{classSingleton}{Singleton$<$\+Your\+Class$>$} to make Your\+Class a singleton. Access the instance using Your\+Class\+::\+Get\+Instance().
\end{DoxyParagraph}
\begin{DoxyParagraph}{Thread Safety\+:}
Uses std\+::call\+\_\+once with std\+::once\+\_\+flag to ensure exactly one instance is created even in multi-\/threaded environments.
\end{DoxyParagraph}
