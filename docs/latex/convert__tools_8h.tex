\doxysubsection{Detailed Description}
High-\/level wrapper functions for character encoding conversions. 

This header provides simplified APIs for common encoding conversion tasks using the \doxylink{classUniConv}{Uni\+Conv} library. These functions serve as a convenience layer for applications that need straightforward UTF-\/8 ↔ wide string conversions without dealing with the complexity of the full \doxylink{classUniConv}{Uni\+Conv} API.
