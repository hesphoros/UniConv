Converts UTF-\/16 encoded string to platform-\/specific wide string format. This function handles the platform differences between Windows (UTF-\/16) and Unix/\+Linux (UTF-\/32) wide character representations.

\begin{DoxyParagraph}{Platform-\/\+Specific Behavior\+:}

\begin{DoxyItemize}
\item {\bfseries{Windows}}\+: Direct assignment (wchar\+\_\+t is 2 bytes, UTF-\/16 compatible)
\item {\bfseries{Unix/\+Linux}}\+: Converts UTF-\/16 to UTF-\/32 using std\+::codecvt\+\_\+utf16
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Algorithm (Windows)\+:}

\begin{DoxyEnumerate}
\item Direct assignment from char16\+\_\+t\texorpdfstring{$\ast$}{*} to wchar\+\_\+t\texorpdfstring{$\ast$}{*} (both are 16-\/bit)
\item No conversion needed as both use UTF-\/16 encoding
\end{DoxyEnumerate}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Algorithm (Unix/\+Linux)\+:}

\begin{DoxyEnumerate}
\item Use std\+::wstring\+\_\+convert with std\+::codecvt\+\_\+utf16 converter
\item Convert UTF-\/16 little-\/endian data to UTF-\/32 wide characters
\item Handle surrogate pairs and extended Unicode characters
\end{DoxyEnumerate}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Use Cases\+:}

\begin{DoxyItemize}
\item Converting Unicode text between different platforms
\item Interfacing with platform-\/specific wide string APIs
\item Processing text data from UTF-\/16 sources (e.\+g., Windows files)
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Error Handling\+:}

\begin{DoxyItemize}
\item May throw std\+::runtime\+\_\+error on invalid UTF-\/16 sequences (Unix/\+Linux)
\item Windows version is safe for valid UTF-\/16 input
\end{DoxyItemize}
\end{DoxyParagraph}

Converts UTF-\/8 encoded string to platform-\/specific wide string format. On Windows, this performs UTF-\/8 → Local → Wide conversion for maximum compatibility with system APIs. On Unix-\/like systems, performs direct UTF-\/8 to wide string conversion.

\begin{DoxyParagraph}{Algorithm (Windows)\+:}

\begin{DoxyEnumerate}

\item Convert locale encoding to wide string using Multi\+Byte\+To\+Wide\+Char
\item Handle errors and edge cases appropriately
\end{DoxyEnumerate}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Algorithm (Unix/\+Linux)\+:}
Direct assignment from UTF-\/8 to wide string (assuming UTF-\/32 wide chars)
\end{DoxyParagraph}
\begin{DoxyParagraph}{Error Handling\+:}

\begin{DoxyItemize}
\item Throws std\+::runtime\+\_\+error on conversion failures
\item Returns empty string for empty input
\item Validates conversion results before returning
\end{DoxyItemize}
