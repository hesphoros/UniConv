\doxysubsection{Detailed Description}
Universal character encoding conversion library. 

\doxylink{classUniConv}{Uni\+Conv} is a comprehensive singleton class that provides bidirectional text encoding conversion between various character encodings including\+:
\begin{DoxyItemize}
\item UTF-\/8 (Unicode Transformation Format 8-\/bit)
\item UTF-\/16\+LE (Unicode Transformation Format 16-\/bit Little Endian)
\item UTF-\/16\+BE (Unicode Transformation Format 16-\/bit Big Endian)
\item GBK (Chinese character encoding)
\item GB2312 (Simplified Chinese character encoding)
\item System locale encodings
\item Various code page encodings
\end{DoxyItemize}

\begin{DoxyParagraph}{Key Features\+:}

\begin{DoxyItemize}
\item Thread-\/safe singleton pattern implementation
\item Comprehensive error handling and reporting
\item Support for both C-\/style and C++ string types
\item Automatic system encoding detection
\item UTF-\/16 endianness handling
\item Memory-\/safe resource management using RAII
\end{DoxyItemize}
\end{DoxyParagraph}
