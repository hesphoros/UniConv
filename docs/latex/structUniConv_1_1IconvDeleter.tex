\doxysubsection{Detailed Description}
Custom deleter for iconv\+\_\+t resource management. 

This RAII deleter ensures proper cleanup of iconv conversion descriptors when used with smart pointers. It safely closes valid iconv descriptors and handles invalid descriptors gracefully.
