\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Singleton$<$ T $>$}
Thread-\/safe \doxylink{classSingleton}{Singleton} pattern template class. 


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & The type to be made singleton\\
\hline
\end{DoxyTemplParams}
Template class that implements the \doxylink{classSingleton}{Singleton} pattern with thread-\/safe lazy initialization. Classes inheriting from this template become singletons with guaranteed single instance creation.

\begin{DoxyParagraph}{Design Pattern\+:}

\begin{DoxyItemize}
\item {\bfseries{Lazy Initialization}}\+: Instance created only when first requested
\item {\bfseries{Thread Safety}}\+: Uses std\+::call\+\_\+once for initialization
\item {\bfseries{RAII Compliance}}\+: Automatic memory management via std\+::shared\+\_\+ptr
\item {\bfseries{Non-\/\+Copyable}}\+: Copy constructor and assignment operator deleted
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Memory Management\+:}

\begin{DoxyItemize}
\item Uses std\+::shared\+\_\+ptr for automatic memory management
\item Instance automatically destroyed when program ends
\item No manual cleanup required
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Thread Safety Guarantees\+:}

\begin{DoxyItemize}
\item \doxylink{classSingleton_afd2f572f321da30e9d331002c92f3956}{Get\+Instance()} is thread-\/safe for initialization
\item Multiple threads can safely call \doxylink{classSingleton_afd2f572f321da30e9d331002c92f3956}{Get\+Instance()} concurrently
\item Exactly one instance is guaranteed to be created
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyWarning}{Warning}
Derived classes should\+:
\begin{DoxyItemize}
\item Make constructor private/protected
\item Declare \doxylink{classSingleton}{Singleton$<$\+T$>$} as friend class
\item Not provide public copy/move constructors
\end{DoxyItemize}
\end{DoxyWarning}
