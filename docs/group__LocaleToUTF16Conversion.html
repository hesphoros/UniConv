Converts UTF-16LE data to the current system's default encoding. This is useful when preparing Unicode data for legacy systems or APIs that expect locale-specific encodings.

std::u16string UniConv::ToUtf16LEFromLocal ( const std::string & input)

Convert system locale encoded string to UTF-16LE.

Parameters
inputInput string in system locale encoding
Returns
UTF-16LE encoded string (std::u16string)

This function converts text from the system's default locale encoding to UTF-16 Little Endian format. UTF-16LE is commonly used on Windows platforms and for data interchange.

Example:
// On a GBK system
std::string gbk_text = "ƒ„∫√ ¿ΩÁ";
std::u16string utf16_text = conv->ToUtf16LEFromLocal(gbk_text);
See also
FromUtf16LEToLocal(), ToUtf16BEFromLocal()
Since
1.0.0.1
Parameters
inputString in system locale encoding
Returns
UTF-16LE encoded string, empty string on error

Converts from the current system's default encoding directly to UTF-16LE format. This bypasses UTF-8 as an intermediate step for better performance and accuracy.

Performance Notes:
  • Direct conversion without UTF-8 intermediate step
  • Optimal for Windows applications using locale data
  • Preserves all characters supported by both encodings
