Detailed Description

Comprehensive test suite for UniConv character encoding conversion library.

This file contains extensive unit tests and integration tests for all conversion functions provided by the UniConv library. It includes tests for system encoding detection, UTF-8/UTF-16 conversions, locale handling, endianness conversion, and error handling scenarios.
