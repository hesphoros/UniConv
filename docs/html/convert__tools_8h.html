Detailed Description

High-level wrapper functions for character encoding conversions.

This header provides simplified APIs for common encoding conversion tasks using the UniConv library. These functions serve as a convenience layer for applications that need straightforward UTF-8 ↔ wide string conversions without dealing with the complexity of the full UniConv API.
