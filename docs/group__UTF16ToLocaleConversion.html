std::string UniConv::FromUtf16LEToLocal ( const std::u16string & input)

Convert UTF-16LE encoded string to system locale encoding.

Convert UTF-16LE string to system locale encoding.

Parameters
inputUTF-16LE encoded string to convert
Returns
String in system locale encoding

This function converts UTF-16 Little Endian text to the system's default locale encoding. Useful for preparing Unicode text for display in legacy applications.

Example:
std::u16string utf16_text = u"Hello World";
std::string local_text = conv->FromUtf16LEToLocal(utf16_text);
See also
ToUtf16LEFromLocal(), FromUtf16BEToLocal()
Since
1.0.0.1
Parameters
inputUTF-16LE encoded string
Returns
String in system locale encoding, empty string on error

Converts UTF-16LE data to the current system's default encoding. This is useful when preparing Unicode data for legacy systems or APIs that expect locale-specific encodings.

Warning
Some Unicode characters may be lost if the target locale encoding doesn't support them. Use with caution for international text that may contain characters outside the system's default character set.
