Parameters

UTF-16LE encoded string (std::u16string)

This function converts UTF-8 text to UTF-16 Little Endian. This is commonly needed for Windows API calls that expect wide character strings.

Example:
std::string utf8_text = "Hello 世界";
std::u16string utf16_text = conv->FromUtf8ToUtf16LE(utf8_text);
See also
FromUtf16LEToUtf8(), FromUtf8ToUtf16BE()
Since
1.0.0.1
Parameters
inputUTF-8 encoded string
Returns
UTF-16LE encoded string, empty string on error

Converts a UTF-8 string to UTF-16 little-endian format. The result is suitable for use with Windows Unicode APIs and other systems that expect UTF-16LE.

Technical Details:
  • Handles all Unicode code points correctly
  • Generates surrogate pairs for characters outside BMP (> U+FFFF)
  • Result has proper little-endian byte ordering
  • Validates UTF-8 input during conversion
// Example: Convert UTF-8 to UTF-16LE for Windows API
std::string utf8_text = "Hello 世界 ?";
std::u16string utf16_text = UniConv::FromUtf8ToUtf16LE(utf8_text);
// Can now use utf16_text with Windows W-APIs
