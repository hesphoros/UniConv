std::wstring UniConv::LocaleToWideString ( const std::string & sInput)

Convert locale-encoded string to wide string.

Convert system locale encoded string to wide string.

Parameters
sInputInput string in system locale encoding
Returns
Wide string (std::wstring) representation

This function converts a system locale encoded string to a wide string using the platform's native wide character encoding. On Windows, this typically results in UTF-16 encoded wide strings.

Platform Differences:
  • Windows: wchar_t is 16-bit, uses UTF-16 encoding
  • Linux: wchar_t is 32-bit, uses UTF-32 encoding
Example:
std::string local_text = "Hello World";
std::wstring wide_text = conv->LocaleToWideString(local_text);

String in system locale encoding, empty string on error

Converts a wide string (std::wstring) to the current system's default character encoding. On Windows, this uses the active code page (ACP). On Unix-like systems, this uses locale-specific conversion.

Platform Implementation:
  • Windows: Uses WideCharToMultiByte() with CP_ACP
  • Unix/Linux: Uses std::codecvt_byname with system locale
Technical Details:
  • Wide string size depends on platform (16-bit on Windows, 32-bit on most Unix)
  • Handles locale-specific character mappings
  • May lose characters not representable in target encoding
