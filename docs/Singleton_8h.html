Detailed Description

Thread-safe Singleton pattern template implementation.

This file provides a thread-safe, modern C++ implementation of the Singleton pattern using std::shared_ptr and std::call_once for guaranteed thread safety and lazy initialization.

Key Features:
  • Thread-safe lazy initialization using std::call_once
  • RAII-compliant using std::shared_ptr
  • Move-only semantics (no copy constructor/assignment)
  • Template-based for type safety
  • Memory-safe automatic cleanup
Usage Pattern:
Inherit from Singleton<YourClass> to make YourClass a singleton. Access the instance using YourClass::GetInstance().
Thread Safety:
Uses std::call_once with std::once_flag to ensure exactly one instance is created even in multi-threaded environments.
// Example usage
class MyClass : public Singleton<MyClass> {
friend class Singleton<MyClass>; // Allow Singleton access to constructor
private:
MyClass() = default; // Private constructor
public:
void DoSomething() { }
};
// Usage
auto instance = MyClass::GetInstance();
instance->DoSomething();
