Detailed Description

Result structure for encoding conversion operations.

This structure encapsulates the result of an encoding conversion operation, including the converted string, error information, and convenience methods for status checking.

Usage Example:
IConvResult result = conv->ConvertEncoding("test", "UTF-8", "GBK");
if (result.IsSuccess()) {
std::cout << "Result: " << result.conv_result_str << std::endl;
} else {
std::cerr << "Error " << result.error_code << ": " << result.error_msg << std::endl;
}
