std::string UniConv::FromUtf8ToLocal ( const std::string & input)

Convert UTF-8 string to system locale encoding.

Parameters
inputInput UTF-8 encoded string
Returns
String in system locale encoding, empty string on conversion failure

This function converts UTF-8 text to the system's default locale encoding. Useful for preparing text for display in legacy applications or for system API calls that expect locale-encoded strings.

Example:
std::string utf8_text = "你好"; // UTF-8 encoded
std::string local_text = conv->FromUtf8ToLocal(utf8_text); // Convert to GBK on GBK system

Converts a UTF-8 string to the current system's default encoding. This is useful when interfacing with legacy APIs or file systems that expect locale-specific encodings.

Common Use Cases:
  • File system operations on non-UTF-8 systems
  • Legacy API compatibility
  • Terminal/console output on Windows
  • Database integration with legacy character sets
Warning

std::string UniConv::ToUtf8FromLocal ( const std::string & input)

Convert system locale encoded string to UTF-8.

Parameters
inputInput string in system locale encoding
Returns
UTF-8 encoded string, empty string on conversion failure

This function automatically detects the system's locale encoding and converts the input string to UTF-8. It's a convenience wrapper around ConvertEncoding() that handles system encoding detection.

Example:
// On a GBK system, convert GBK string to UTF-8
std::string gbk_text = "你好"; // Assumes this is GBK encoded
std::string utf8_text = conv->ToUtf8FromLocal(gbk_text);
See also
ConvertEncoding(), GetCurrentSystemEncoding()
Since
1.0.0.1
Parameters
inputString in system locale encoding
Returns
UTF-8 encoded string, empty string on error

Converts a string from the current system's default encoding to UTF-8. The system encoding is automatically detected using GetCurrentSystemEncoding().

Platform Behavior:
  • Windows: Converts from active code page (GBK, Big5, etc.) to UTF-8
  • Unix/Linux: Usually no conversion needed (UTF-8 → UTF-8)
// Example: Convert GBK string to UTF-8 on Chinese Windows
std::string gbk_text = "你好世界"; // In GBK encoding
std::string utf8_text = UniConv::ToUtf8FromLocal(gbk_text);
