std::u16string UniConv::FromUtf16LEToUtf16BE ( const std::u16string & input)

Convert UTF-16LE string to UTF-16BE.

Parameters
inputUTF-16LE string to convert
Returns
UTF-16BE encoded string (std::u16string)

This function converts between UTF-16 endianness formats. The text content remains the same, but the byte order changes from little-endian to big-endian.

Technical Details:
  • Swaps byte order of each 16-bit code unit
  • Preserves Unicode code points and text meaning
  • Handles surrogate pairs correctly
See also
FromUtf16BEToUtf16LE()
Since
1.0.0.1
Parameters
inputUTF-16LE encoded string
Returns
UTF-16BE encoded string, empty string on error

Converts between UTF-16 little-endian and big-endian formats. This is essential for cross-platform data exchange where different systems use different byte ordering.

Technical Details:
  • Performs byte order conversion without changing Unicode content
  • Handles surrogate pairs correctly
  • Preserves all Unicode code points
  • Uses efficient iconv-based conversion
Use Cases:
  • Network protocol data exchange
  • Cross-platform file format conversion
  • Java/C# interoperability
  • Database migration between different endianness systems
